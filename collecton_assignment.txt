Develop a Java application Student and Department Management System using the Java
Collection Framework. Requirements:

1. The system should maintain the list of departments using:
Set<Department> departments = new HashSet<>();

Duplicate departments should not be allowed based on Department ID.

2. The system should maintain the list of students in admission order using:
List<Student> students = new LinkedList<>();

3. The system should maintain student records for quick searching using:
Map<Integer, Student> studentMap = new HashMap<>();

Key = Roll Number, Value = Student Object.

Department
• deptId
• deptName

Student
• rollNo
• name
• marks
• Department department
