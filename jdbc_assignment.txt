Develop a menu-driven Employee Management System using Java, JDBC, and MySQL to manage employee records stored in a database. The application should allow users to perform various database operations such as inserting, updating, deleting, searching, and displaying employee information.


Employee Contains:

empId
empName
department
salary

===== Employee Management System =====

1. Add Employee
2. Update Employee
3. Delete Employee
4. Search Employee
5. Display All Employees
6. Exit
